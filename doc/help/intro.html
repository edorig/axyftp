Introduction

What is AxY FTP

AxY FTP is an X Window System FTP client designed to be used mainly on Linux workstations. It is written using Motif widget set and XmAxyList Widget. GTK+ GUI implementation is also available.

Current version of AxY FTP is just a "code escape". Only quite minimal set of features is implemented, with several known bugs pending. Users are encouraged to report desired features and found bugs to the author.

Features

Major features available in current (0.4.4) release:

  • user-friendly, intuitive interface (both GTK+ and Motif).
  • passive and non-passive data transfers
  • convenient session manager window
  • on-line help system
  • recursive downloads with resume option
  • recursive uploads
  • multiple delete with option to recurse
  • auto-redial option
