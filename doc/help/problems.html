Known Problems

Current release of AxY FTP is supposed to have quite a few problems. Here are some of them that I am aware of:

  • "Cancel" button does not work reliably.
  • Some sites appear without directory listing.
  • Some firewalls are not supported
  • Directory listing appears unsorted.
  • VMS-style directory listing is not supported

Please report the problems you encounter to Alexander Yukhimets, the author of AxY FTP.
